description: Runnable examples showing common Delta-MCP patterns.

All examples live in [`examples/`](https://github.com/norhther/delta-mcp/tree/main/examples) and run with `npx tsx`.

| Example | What it shows |
|---------|--------------|
| [stdio-server](/delta-mcp/examples/stdio-server/) | Minimal server with two tools — start here |
| [client-usage](/delta-mcp/examples/client-usage/) | `DeltaClient` connect, progressive disclosure, on-demand schema fetch |
| [result-handler](/delta-mcp/examples/result-handler/) | Truncation, pagination, object summarization, rate-limit handling |
| [filesystem-server](/delta-mcp/examples/filesystem-server/) | Real-world server: read/write/list/search files, path sandbox |
| [pagination](/delta-mcp/examples/pagination/) | 1 000-record server, 6 pagination patterns including auto-fetch-all |
| [http-oauth-server](/delta-mcp/examples/http-oauth-server/) | HTTP + OAuth 2.1 resource-server, RS256, curl walkthrough |
